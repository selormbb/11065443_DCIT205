UG Computer Science Department

Job opportunities

  • Software Developer/Engineer
    • Design, develop, and test software applications. Specializations include web development, mobile app development, and software engineering.
  • Data Scientist/Data Analyst
    • Analyze and interpret complex data to aid in decision-making. Data scientists use machine learning and statistical methods to derive insights.
  • Cybersecurity Analyst
    • Protect systems, networks, and data from cyber threats. This involves creating security measures and responding to security breaches
  • Database Administrator
    • Manage and maintain databases, ensuring they function efficiently and securely. This role involves organizing data, troubleshooting issues, and implementing security measures.
  • IT Project ManagerOversee
    • Oversee projects related to technology implementations, ensuring they are completed on time and within budget. Requires leadership, organizational, and technical skills
  • AI/Machine Learning Engineer
    • Develop algorithms and systems that enable computers to learn and make decisions. This involves programming, data analysis, and knowledge of machine learning techniques.
  • Computer Systems Analyst
    • Evaluate an organization's computer systems and procedures to improve efficiency and effectiveness. They analyze user requirements, procedures, and problems to automate or improve existing systems.
  • Network Engineer
    • Design and implement networks, ensuring they operate efficiently and securely. This includes troubleshooting network issues and optimizing performance.
  • Cloud Architect/Engineer
    • Design and manage cloud infrastructure and services, ensuring scalability, security, and reliability of applications hosted on the cloud.
